  • More data exported from non-conformance reports

New RFQ module works from MRP, BOM and parts list to raise an RFQ for any part. Invite multiple suppliers, direct email and then convert the winning supplier to a PO as we automatically close the other RFQs.

For users of Intouch Monitoring, you can now update the quantities completed during a shift by imported the update .csv file from your Intouch system.

Price lists and Price Break Quantities

The bill of materials (BOM) record now has an updated price list feature to allow for price break quantities to be created based on the pricing model you use. Once the price list is set up, sales order line items will read the correct price based on the quantity requested on the contract line item.

06/2025
Categories and Type Update

Part type has been changed to a simple button selection between manufactured and purchased parts. This now means categories are available for both manufactured products and purchased products.

06/2025
SFDC Updates

Our SFDC (Shop Floor Data Capture) module has been updated with an easier workflow, access to work order documents from the SFDC page and the ability to add notes to SFDC which are then recorded on the work order operation.

06/2025
Infinite Capacity Workcenter

Any workcenter can be marked as having "infinite capacity", meaning multiple jobs can be scheduled simultaneously, perfect for batch processing like painting, heat treatments, cooling off. On our drag and drop scheduling tool, users can now overlay operations on top of each other for infinite capacity workcenters.

05/2025
Work Order Cost Analysis and Variance Report

Each work order now includes a cost breakdown for parts and labour, and for users of our advanced plan, this is summarised in a cost variance report.

05/2025

SFDC analytics gives users insights into workcenter and employee booked hours and ulitisation and also shows total booked hours by operation.

05/2025

When sending by email or printing order acknowledgements, a new tracking status will be displayed on the contracts.

04/2025
Labour utilisation and composite costing of operations

Operations on the bill of materials can be a single cost or a composite of labour and workcenter cost. For pro and moulding users, labour utilisation can also be set on each operation.

03/2025
Foreign currencies update

Foreign currencies are now converted using live market rates. All KPIs run on base rate currency, and purchasing, invoicing and sales can be easily entered in either your base currency of the foreign currency. Foreign currencies can also be used in the BOM parts.

02/2025

Each part record now has a view of all related BOMs so you can see where any part in the system has been used in a BOM.

01/2025

Works Order Box Labels

When packaging large volume parts from production runs into boxes, this new box labelling features allows each box to be identified by the quantity of parts within the box. The system will automatically calculate the labels needed base on the parts per box, defined in the Bill of Materials.

Packaging Items

Parts can be marked as "packaging" which allows users to define how many parts will be packaged into each container. After entering the quantity the price for the BOM will be divided by the "parts per box".

Data import tools are now added to the parts, customer and supplier listings. This enables users to upload their own data from Excel spreadsheets.

11/2023

A new page called Key Performance Indicators is now released. This moves the OTD from the dashboard to the KPI page and makes space for task management. The KPI page is now also updated with invoices sales and order intake metrics.

10/2023
ERP Task Management

New feature released! Tasks management tool for all users of Fraction ERP to manage anything - choose your own categories such as maintenance, purchasing, 5S, quality audits and customer follow ups. We will be integrating this feature with the data in Fraction to make it even more powerful.

08/2023
Certificate of Conformance

All shipments records can now be used to create a certificate of conformance. This feature includes a scanned signature to add to the certificate along with your own customised declaration.

06/2023
New tooling database to specify jigs, tools and fixtures required for any BOM

Any jigs or tooling used during production can be stored in the tooling database. This list is then available to add the tooling or jigs required to each BOM and the tooling list is shown on the route card. This can even be applied for cutting tools. This is available on pro and moulding versions.

06/2023
Scrap items are logged when completing a WO

If some parts are scrapped during production, this can be logged when completing a WO - meaning all parts are consumed, but only the completed quantity is entered into stock.

06/2023
Production yield used to determine WO quantities for production

Any part can now contain a scrap rate. This is used to determine how many parts should be requested on the works order in order to meet the required quantity and allow for the scrap rate. This is available on pro and moulding versions.

06/2023
Use percentage to specify the part ratios used in BOMs for plastics and chemicals

Parts on a BOM can be defined as a percentage of the BOM part. The part record contains the part mass, which is used when calculating the requirements for the WO part demand (MRP). This is available on pro and moulding versions.

Raise purchase orders directly from the sales order line item

If you are buying in parts for your sales contracts you can now raise a PO directly from the sales order line.
